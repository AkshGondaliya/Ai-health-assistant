# AI Health Assistant

AI-powered health tracking application that predicts obesity levels using Machine Learning and provides personalized recommendations.

## Features

- JWT Authentication
- Obesity Prediction
- BMI Calculation
- Health Recommendations
- Analytics Dashboard
- Prediction History
- User Profile Management

## Tech Stack

Backend:
- FastAPI
- PostgreSQL
- SQLAlchemy

Frontend:
- React
- Vite
- Axios

Machine Learning:
- Scikit-Learn
- Random Forest

Database:
- Neon PostgreSQL
